# 🧀 Cheese Dash!

A cute 3D **mouse vs cat** game you can play in your web browser — on a computer or a phone!

- 🐭 You are the mouse. Grab **12 cheese** to win.
- 🐱 Run away from the hungry cat — it gets faster the more cheese you grab!
- 💻 **Computer:** use the arrow keys or W A S D.
- 📱 **Phone/tablet:** use the touch joystick in the corner.

Made with [Three.js](https://threejs.org). It's all in one file: `index.html`.

## Play it

Just open `index.html` in a web browser, or visit the live link (GitHub Pages).
